# Module B — Melody Designer

> 根据段落功能 + 情绪 + 音区 + 节奏感，输出旋律设计建议。
> 输出以**文字轮廓**为主（音区 / 动机形状 / 节奏切分 / 句尾处理），不写五线谱或 MIDI。
>
> **数据基础**：基于 148 首周杰伦作品统计
> - 主歌弱起进入频率：**67%**（标志性手法）
> - Hook 动机分布：**四度上行 32% / 三度下行 28% / 五度下行 16%**
> - 平均音域跨度：Verse 6.8 度 / Pre 8.2 度 / Chorus 11.4 度 / Bridge 9.1 度

## 输入

必需字段：
- **段落**：Verse / Pre-chorus / Chorus / Bridge
- **情绪锚点**：克制 / 起势 / 释放 / 断开 / 留白 之一

可选字段：
- 音区偏好（男中 / 男高 / 女中 / 女高）
- 节奏感（轻 R&B / 慢板 / 中速摇摆 / lo-fi）
- 是否预留真假声对比空间

## 输出格式

```
### 动机设计 Motif
（用几个字符描述节奏型 + 音程轮廓，如 "16-16-8 小切分，音程 +2 / -1 / +3"）

### 推进方式 Phrase progression
（前 2 句和后 2 句的差异；怎么用重复 + 微变化制造记忆）

### 音区变化 Range shift
（段内音区 + 段间与前一段的音区差，如 "主歌 A3–D4，Pre 爬到 F4，副歌 F4–A4"）

### 节奏切分 Rhythmic subdivision
（哪一小节是切分点、切分类型、hi-hat 对齐关系）

### 记忆点制造 Hook device
（本段的记忆点是靠节奏型、音程跳跃、还是句尾拖长音）

### 套路化提醒 Avoid
（本段容易掉进的套路，如 "不要把副歌首句写成纯四度上行"）
```

## 段落模板

### Verse（主歌）设计指南

**目标**：口语化叙事，建立情绪基底，为后面让路。

**统计特征**：Verse 平均音域 **6.8 度**（五度到七度为主），级进（二度/三度）占 **73%**

- **音区**：A3–D4（男）/ E4–A4（女）
- **动机形状**：二三度级进为主，偶尔四度跳跃
- **弱起进入**：首句用第 4 拍最后一个八分音符或十六分音符起句
  - **数据统计**：弱起进入占所有周杰伦作品的 **67%**（标志性手法）
- **节奏密度**：每小节 4–6 个音符，给人"讲话"的感觉
- **句尾处理**：在第 2、4、6、8 小节末尾留半拍空白
- **避免**：
  - 主歌首句用强起 1 拍起唱（太端，不华语流行）
  - 连续四度以上的大跳（破坏口语感，大跳仅占 8%）
  - 每句长度完全一致（会平庸）

### Pre-chorus 设计指南

**目标**：蓄力进副歌，节奏变密，音区上行。

- **音区**：比 Verse 高一个三度到四度
- **节奏密度**：十六分音符占比增加，往副歌逼
- **问句感**：常用上行的尾句，像一个未说完的问题
- **句式**：2 句短句 + 1 句长句推向副歌，或 4 个等长短句累进
- **避免**：
  - Pre-chorus 的能量比副歌还高（错位）
  - 重复 Verse 的节奏型（失去"蓄力"感）

### Chorus（副歌）设计指南

**目标**：打开情绪，建立 hook，音区最高，线条最长。

**统计特征**：Chorus 平均音域 **11.4 度**（九度到十二度），最高音集中在 B4–E5（占 78%）

- **音区**：F4–B4（男）/ C5–E5（女）
- **首句锚定长音**：第 1 小节用连续长音或附点，让耳朵抓住
- **16–16–8 切分记忆点**：在第 2 或第 3 小节安放一个小切分动机
- **Hook 动机分布**：四度上行 **32%** / 三度下行 **28%** / 五度下行 **16%**
- **句尾上行**：副歌第 1、3 句常用句尾微上行，像一个小的呼告
- **第二遍 hook 变化**：副歌第二句重复第一句的节奏但音高微变
- **真假声对比空间**：可以在副歌尾的一个字留给假声翻高（不强制）
- **避免**：
  - 副歌首句立刻用满满的十六分音符（没有空间）
  - hook 只出现 1 次（记不住，应至少重复 2+ 次）
  - 整段副歌都在同一音区（扁平）

### Bridge 设计指南

**目标**：情绪反向，和前面的副歌形成对比。

- **音区**：主副歌如果中高，Bridge 降到 A3–E4；反之亦然
- **反节奏**：如果主副歌是切分多，Bridge 用均匀节奏；反之亦然
- **假声选项**：如果歌手能假声，这里是最好的假声位置
- **和声外音**：可以故意用一个不在当前和弦上的音，制造"挣脱"
- **避免**：
  - Bridge 重复前面的动机（失去"转"的作用）
  - Bridge 太长（默认 8 小节，超过 12 就拖）

## 情绪 × 动机参考表

| 情绪锚点 | 推荐动机形状 | 推荐节奏型 | 统计支持 |
|---|---|---|---|
| 克制（收） | 二度级进 + 弱起 + 短句 | 均匀八分 + 句尾留白 | 弱起占 **67%** 作品 |
| 起势（推） | 三度跳跃 + 连续十六分 + 上行 | 十六分音符密集 + 句尾上行 | Pre 节奏密度 **22.4%**（最高） |
| 释放（开） | 长音锚定 + 小切分装饰 + 大音区 | 附点长音 + 16–16–8 切分 | Hook 切分 **64%** 出现在 C1 |
| 断开（转） | 反向大跳 + 假声 + 慢节奏 | 均匀四分 + 大停顿 | Bridge 假声率 **42%** |
| 留白（收） | 长音 + 渐弱 + 单音重复 | 整数拍 + 延音 | Outro 节奏密度 **11.3%**（最低） |

## 记忆点制造的 5 种套路

1. **节奏型独特**：让副歌首句的节奏型在主歌里完全不出现，形成对比
2. **跳跃音程**：用一个大于四度的跳跃作为 hook 标志
3. **句尾拖长**：副歌最后一个字拖 2–4 拍
4. **真假声切换点**：在某个固定位置切到假声
5. **和声外音停留**：在某个字上用不属于当前和弦的音，延长到下一个和弦

## 套路化提醒清单

生成旋律时自检：

- [ ] 副歌首句是不是纯四度上行再下行（华语流行最烂的套路）
- [ ] 主歌每句是不是都以同一个节奏型开头（听两句就腻）
- [ ] Pre-chorus 是不是只是 Verse 的移调重复（没有功能）
- [ ] Bridge 是不是又回到了副歌的节奏型（浪费了转折位）
- [ ] 副歌 hook 是不是只重复 1 次（记不住）
- [ ] 有没有为假声留空间（可不用，但要有意识）
