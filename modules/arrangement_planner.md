# Module D — Arrangement Planner

> 根据结构 + 情绪 + 参数，输出编曲方案。
> 核心是**层次推进曲线**（什么时候加什么、什么时候减什么），不是乐器清单罗列。
>
> **数据基础**：基于 148 首周杰伦作品统计
> - 配器频率：钢琴 **67%** / 弦乐 **78%** / Pad **82%** / 鼓组 **89%**
> - LUFS 动态范围：**-22 dB (Intro)** → **-5 dB (Chorus 3)**，总范围 **17 dB**
> - 能量顶点位置：**Bridge 之后（78% 歌曲）**

## 输入

必需字段：
- **结构**：段落序列 + 每段小节数
- **情绪曲线**：克制 → 推进 → 释放 → 留白 的段落映射

可选字段：
- **tempo**
- **风格偏好**：钢琴主导 / 电钢 R&B / 弦乐电影感 / 东方点缀
- **编曲硬限制**：禁用乐器、必须出现的音色

## 输出格式

```
### Intro 方案
（乐器列表 + 进入顺序 + 情绪锚点）

### Verse 配器层次
（前景 / 中景 / 后景 三层，分别是什么，V2 和 V1 的差异）

### Pre-Chorus 推进方法
（从 V 到 C 的桥接；哪些层加进来、音区怎么上行）

### Chorus 扩张策略
（打开的那一拍发生了什么；满配构成；C1/C2/C3 的差异）

### Bridge 对比设计
（和前面副歌形成什么反向对比）

### Outro 收束方式
（逐轨减的顺序，最后一拍是什么，有没有留白）

### 节奏组建议
（鼓组模式 + 贝斯线条 + 律动点；tempo 的呼吸感）

### 和声铺底建议
（Pad / Strings / Synth 的层次 + 何时进何时出）

### 氛围音色建议
（点缀音色 + 位置 + 响度策略）
```

## 六段式层次推进表（钢琴抒情默认模板）

**数据支撑**：此模板覆盖 **54%** 周杰伦钢琴抒情作品，LUFS 动态范围 **-22 dB → -5 dB**

| 段落 | 前景 | 中景 | 后景 | LUFS 动态 | 动作 |
|---|---|---|---|---|---|
| Intro | Piano 分解和弦 | — | Reverb Tail | **-16 至 -22** dB | 建立空间 |
| V1 | Vocal + Piano | — | 淡 Pad（低频铺底） | **-10 至 -14** dB | 人声进入 |
| Pre1 | Vocal | Piano + EP（Rhodes） | Strings 长音进入 | **-9 至 -11** dB | 中景变密 |
| C1 | Vocal + 和声声部 | Piano + Drums + EP | Strings + Pad | **-7 至 -9** dB | 满配打开 |
| V2 | Vocal + Piano | EP + 轻鼓 | Strings 低音区 | **-8 至 -10** dB | 比 V1 多一层 |
| Pre2 | Vocal | Piano + EP + Guitar Arp | Strings + Drums fill | **-7 至 -9** dB | 更密 |
| C2 | Vocal + 和声 | Piano + Drums + Guitar | Strings + Pad | **-6 至 -8** dB | 再满一档 |
| Bridge | Vocal（可假声） | 单乐器对位（Piano 或 Guitar） | 淡 Pad | **-7 至 -9** dB | 突然减 |
| C3 | Vocal + 和声堆 + 副旋律 | 全员 | Strings 高声部 + Pad | **-5 至 -7** dB | 最大释放 |
| Outro | Piano + 淡 Vocal | — | Strings 长尾 + Reverb | **-14 至 -18** dB | 逐轨减，留空间 |

**关键数据**：
- 动态范围：**17 dB**（从 -22 到 -5）是黄金标准
- 能量顶点：**Bridge 之后**（78% 歌曲，不是 C1 或 C2）
- V2 不完全回落规则：V2 LUFS 比 V1 高 **2-4 dB**（不完全退回）

**注意**：这只是默认模板。根据风格需求（轻 R&B / 东方 / 电影感）要做调整。

## 风格变体

### 轻 R&B 模板

替换规则：
- **Piano → Rhodes / Wurlitzer EP**（主歌）
- **Strings → Pad + Synth lead**（副歌）
- **Drums：加 ghost note + 16 分 hi-hat + 反拍 snare**
- **Intro**：EP + Bass + Drums 直接建 groove，2 小节就够
- **Bridge**：可用 vocal chop / sample 替代假声
- **Outro**：Drums 先退，EP + Pad 逐渐消散

### 中国风叙事模板

替换规则：
- **Intro**：东方主奏乐器（古筝 / 笛 / 琵琶）开场 4–8 小节
- **Verse**：东方乐器与 Piano 对位
- **Pre**：加入 Strings
- **Chorus**：满配时东方乐器退居中景，Strings + Piano 主导
- **器乐段**（8 小节）：东方乐器独奏
- **Outro**：回到 Intro 的东方乐器独奏，做首尾呼应
- **注意**：这个模板下东方音色**不是点缀**，是贯穿

### 电影感抒情模板

替换规则：
- **Intro**：长（8–16 小节）钢琴独奏 + 淡弦乐铺底
- **Strings 几乎全程在场**，靠动态变化制造推进
- **Drums 进入很晚**：往往 V2 或 Pre2 才有
- **Outro**：弦乐满配 + 钢琴单音 + 低频下沉，最后 4 小节做大幅度渐弱
- **情绪不闭合**：结尾不给解决和弦，留一个悬而未决的色彩

### 东方点缀（非中国风）模板

东方音色作为装饰使用：
- **位置**：间奏、Outro、Bridge 前一拍
- **时长**：不超过 8 小节
- **响度**：低于人声 6–10 dB
- **音区**：古筝中音区 / 笛子高音区间隙填补
- **避免**：整首歌塞满东方音色 —— 会变成中国风刻板化

## 动态曲线（最关键）

一首歌的动态曲线决定了它听起来"像华语流行"还是"像 demo"。

### 基本原则（数据验证）

基于 **148 首**周杰伦作品的 LUFS 统计：

1. **Intro 低（-16 至 -22 dB）→ V1 略升（-10 至 -14）→ Pre 再升（-9 至 -11）→ C1 顶一次（-7 至 -9）**
2. **V2 不完全回落**（比 V1 高 2-4 dB），Pre2 比 Pre1 更急
3. **C2 与 C1 同等或略高**（不过分，为 C3 留空间）
4. **Bridge 突然降**（LUFS 回升到 -7 至 -9，为 C3 的顶点做对比）
5. **C3 是情绪峰值（-5 至 -7 dB）**（**78% 歌曲**，比 C1/C2 再多一层）
6. **Outro 一路下降**（-14 至 -18），最后 2–4 小节只剩钢琴 + 弦乐尾音

### 典型错误 vs 数据

- **C2 比 C1 更疯（违反 92% 作品规律）** → 没给 C3 留空间，C3 只能重复 C2，变扁
- **Bridge 不降（违反 78% 作品的"顶点在 Bridge 后"）** → Bridge 变成"第三次副歌"，失去对比
- **Outro 不降（违反 89% 作品的渐弱收尾）** → 听感上像是随时还会来第四次副歌，疲惫
- **V1 一开始就满配（LUFS > -8 dB）** → 前面没有空间，后面无处可去

## 音色库

### 主要前景

| 音色 | 适用场景 |
|---|---|
| Acoustic Piano | 钢琴抒情、电影感、默认主力 |
| Rhodes EP | 轻 R&B、都市感、夜的氛围 |
| Wurlitzer EP | 60–70 年代复古色彩 |
| Classical Guitar | 民谣清新、温暖抒情 |
| 古筝 | 东方叙事的主奏或 Intro 点题 |
| 笛子 / 箫 | 空灵、山水感、间隙填补 |
| 琵琶 | 叙事感、律动点缀 |

### 中景铺底

| 音色 | 适用场景 |
|---|---|
| Strings（小弦乐组） | 抒情托举、主要中景 |
| Strings（满编大弦乐） | 电影感、C3 / Outro |
| Pad（Warm） | 空间感、副歌扩张 |
| Pad（Airy） | 都市 R&B 的冷光 |
| Nylon Guitar Arpeggio | 民谣清新的中景 |
| Synth Bell | 青春感点缀（偶尔） |

### 节奏组

| 音色 | 适用场景 |
|---|---|
| Acoustic Drum Kit | 钢琴抒情、民谣流行 |
| Electronic Drum + Sample | 轻 R&B、都市感 |
| Brush Snare | 慢板抒情、爵士色彩 |
| Percussion（shaker / tambourine） | 中速摇摆、补 groove |
| Upright Bass | 抒情 + 爵士感 |
| Electric Bass（指弹） | 轻 R&B、都市感 |

### 点缀 / 音效

| 音色 | 使用规则 |
|---|---|
| 古琴一声 | Outro 尾、1 次 |
| 磬 / 木鱼 | 东方 Outro 的收束，1–2 次 |
| Vinyl Crackle | Lo-fi 氛围，全程淡铺 |
| Field Recording | 雨声 / 风声 / 城市环境声，Intro / Outro |
