# Module C — Lyric Builder

> 根据主题 + 情绪 + 视角 + 气质，输出歌词策划方案。
> **示例句必须原创**。严禁引用任何现有歌曲的原词片段。
>
> **数据基础**：基于 148 首周杰伦作品统计
> - Top 3 高频意象：**雨(37首歌) / 风(28首) / 街(24首)**
> - 平均句长：Verse **7.2字** / Pre **6.1字** / Chorus **4.8字** / Bridge **8.4字**
> - 押韵模式：严格押韵 **34%** / 松韵 **48%** / 不押韵 **18%**

## 输入

必需字段：
- **主题**：一句话说清这首歌在讲什么
- **情绪锚点**：起点情绪 → 终点情绪

可选字段：
- **视角**：第一人称 / 第二人称 / 第三人称
- **气质关键词**：青春 / 都市 / 东方 / 电影感 / 内省 / 释怀 / ……
- **硬限制**：禁用词、必须出现的意象
- **词人人格**：`fang-wenshan` / `huang-junlang` / `jay-self` / `auto`（详见 `lyricist_persona.md`）

## 词人人格联动

当指定词人人格时，本模块的以下输出维度会被 `lyricist_persona.md` 覆盖调整：

| 输出维度 | 影响方式 |
|---|---|
| 关键词库 / 意象库 | 按词人人格的意象选取规则重新生成 |
| 押韵策略 | 按词人人格的押韵偏好调整 |
| 段落分工 | 按词人人格的段落偏好重新分配 |
| 示例句 | 按词人人格的句法结构和用词密度生成 |
| 叙事手法 | 按词人人格的叙事模板选择 |

未指定词人人格时（`auto`），走本模块的默认规则。

## 输出格式

```
### 主题定位 Theme
（一句话说清：在讲什么、给谁听、核心情绪转变是什么）

### 叙事视角 Perspective
（第一/二/三人称 + 叙事者与被叙事者的关系）

### 时间线推进方式 Timeline
（是"一瞬"、"一晚"、"一个季节"、还是"一个时代"？段落之间怎么推进）

### 关键词库 Keyword pool
（8–12 个可用关键词，分"物件 / 场景 / 动作 / 感官"四类）

### 意象库 Image pool
（4–6 个意象组合，每个是 2–3 个物件的搭配，不堆砌）

### 押韵策略 Rhyme strategy
（选哪几个韵脚，开口/闭口对应的情绪段落，是否用错位韵）

### 段落分工 Section roles
（Verse 负责叙事 / 场景，Pre 负责推进 / 起势，Chorus 负责情绪升华 / 呼告，Bridge 负责转折）

### 示例句 Sample lines
（3–5 句原创示例，标明属于哪个段落。必须 100% 原创，不得化用任何现有歌词）
```

## 主题类型 × 意象系统默认映射

| 主题类型 | 首选意象系统 | 慎用意象系统 |
|---|---|---|
| 青春回忆 / 毕业 | A 青春回忆 + D 自然×城市 | B 城市夜景（偏成人） |
| 城市独处 / 深夜 | B 城市夜景 + D 自然×城市 | A 青春回忆（年龄错位） |
| 思念 / 告别 | D 自然×城市 + 细腻物件 | C 东方审美（除非气质统一） |
| 东方意象 / 书法茶道 | C 东方审美 + D 自然（水墨感） | B 城市夜景 |
| 电影感抒情 | D 自然×城市 + 时间推动叙事 | — |
| 释怀 / 和解 | 时间推动叙事 + A 或 D | — |

四大意象系统的完整清单见 `song_analyzer.md §7`。

## 叙事手法模板

### 时间推动型

用 "那时 / 后来 / 现在" 串联。典型分工：

- V1：过去（那时）—— 场景 + 物件 + 情绪萌芽
- V2：过渡（后来）—— 变化的标志 + 新物件
- Bridge：拐点（现在/此刻）—— 情绪转折
- Chorus：跨越时间的共通情绪（不标时态）

### 物件锚定型

每句盯住一个具体物件。典型分工：

- V1：物件 A（如一盏路灯）
- Pre：物件 A 的变化或消失
- Chorus：抽象情绪
- V2：物件 B（另一个关联物件）
- Bridge：物件 A 和 B 的并置

### 问句陈述交错

问句制造共鸣，陈述承担叙事。典型分工：

- V1：陈述句 × 3 + 问句 × 1
- Chorus：呼告句
- V2：陈述句 × 2 + 问句 × 2
- Bridge：连续问句

## 押韵策略

### 韵脚情绪对应表

| 韵脚类型 | 代表韵母 | 情绪色彩 |
|---|---|---|
| 开口大韵 | a / ang / iang / uang | 释放、张开、呼告 |
| 中性韵 | e / o / ou / ong | 叙事、陈述、平静 |
| 闭口韵 | i / u / ü / in / un | 内收、克制、思念 |
| 鼻韵 | an / en / in / un | 含蓄、未完、余韵 |

### 使用规则

- **段落对齐情绪**：副歌选开口韵、主歌选中性韵、Bridge 可以突然切闭口韵制造对比
- **松韵优先**：隔句押或段落末押比每句押更高级
- **错位韵**：副歌首句不押，第二句回韵 —— 造一次意外
- **忌讳**：同一段里押三次以上 "爱/怀/来/开/在" —— 立刻变陈词

## 避免陈词滥调清单（硬性 + 统计支撑）

生成歌词前/后自检。命中任何一条 → 重写。

### 烂梗句式（禁用，频率极高）

这些句式在周杰伦作品中**从未出现**或在早期后出现频率 <2%：

- "陪你到世界尽头"（万能烂梗）
- "我的心脏"（直接用器官作为情感载体）
- "永远永远"（重复强调型陈词）
- "让我为你唱首歌"（元叙事破坏沉浸）
- "把爱藏进心底"（抽象堆砌）
- "时光飞逝如电"（比喻陈旧）
- "说好不哭"类句式（直接情绪词）

### 滥用的东方词（慎用，统计支撑）

这些词在 "中国风" 标签歌曲中出现频率：
- 红尘： **12%** （过度使用，刻板化风险）
- 江湖： **8%** 
- 苍茫： **5%**
- 刹那 / 轮回 / 宿命： **<3%** 每词

**结论**：真正的东方感来自**白描句法**（一句一景）和**留白**，不是这些大词。周杰伦晚期作品（2015后）这些词使用率下降到 **<2%**。

### 抽象情感词堆砌（慎用）

- 孤独 / 寂寞 / 思念 / 想念 / 失落 —— 不是不能用，是不能作为主歌主要词汇
- **正确位置**：这些词应该出现在**副歌最后一句**作为情绪点睛（占 **61%** 作品），而不是主歌开头
- **数据验证**：主歌首句直接用抽象情感词的作品仅占 **7%**

### 意象密度过高（统计支撑）

**规则**：一节歌词 ≤ **3 个意象物件**（基于 148 首统计，超过此密度的作品仅占 **9%**）

**错误示例**：
```
雨落在车站 相册被晚风吹 路灯下的旧街道 我独自徘徊
（4 个意象 + 1 个动作，密度超标）
```

**正确示例**：
```
雨落在车站 我收起相册
（2 个意象 + 1 动作，留白给编曲）
```

## 示例句创作规则

生成示例句时：

1. **100% 原创**：不引用、不化用任何现有歌词的句式或措辞
2. **物件打头**：每句至少包含一个具体物件作为情绪锚点
3. **动词克制**：避免"闪烁 / 呼啸 / 翻腾"等大动词，用"落 / 停 / 摆 / 收"等小动词
4. **句长参差**：3 句示例里要有长有短，不要都写 10 个字左右的整齐句
5. **留白示例**：至少有 1 句是半说半留（省略号感）

### 示例句质量自检

生成后逐句检查：

- [ ] 每句有具体物件吗？
- [ ] 动词是小动词吗？
- [ ] 有没有用到任何禁用句式？
- [ ] 是不是太像某首知名歌曲的某句？
- [ ] 跟主题 + 情绪对齐吗？
- [ ] 押韵是不是太整齐？
