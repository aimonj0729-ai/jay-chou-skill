# Test 03 — 东方意象 + 陈词回避

## 用途

验证 Skill 在处理东方主题时**不会滑进"红尘/江湖/苍茫"等懒人东方化**，能用白描句法实现真正的东方感。

## 输入

```
写一首关于茶、书法、晚上独自练字的歌，要有东方审美的气质。
```

## 预期行为

Skill 应当：

1. 识别这是东方主题（主题 + 气质都明示）
2. 从 `song_analyzer.md §7` 调用系统 C（东方审美）+ 白描句法
3. 歌词示例句**不得出现**下列词：红尘、江湖、苍茫、刹那、轮回、缘分、宿命、千年、万载、一梦、流年（这 10 个是东方万能词黑名单）
4. 编曲可用东方音色（古筝/笛/琵琶），但要按 `arrangement_planner.md §东方点缀规则`：**点缀而非贯穿**（除非 Skill 主动选择升级为中国风叙事模板，此时可以贯穿）
5. 情绪要细，不要"大志向"（不写"我要挥毫千古"这种）

## 验收清单

### 必须项

- [ ] 10 段完整输出
- [ ] 歌词示例句 **0 次** 命中东方万能词黑名单
- [ ] 示例句是白描句法（一句一景 / 一笔一物），不用大形容词
- [ ] 动词是小动词（研 / 收 / 落 / 停 / 铺 / 洗），不是大动词（挥 / 泼 / 翻腾 / 奔流）
- [ ] Similarity Guard 的"歌词陈词滥调"一项显式给 PASS 或 WARN 并说明原因

### 应当项

- [ ] 意象库里出现具体物件：墨 / 砚 / 宣纸 / 一盏灯 / 檐 / 茶杯的痕迹 / 镇纸 / 窗纱 等
- [ ] 情绪是"内省 / 独处 / 缓慢"，不是"豪情 / 励志 / 怀古"
- [ ] 东方音色的使用时长与方式符合"点缀"或"贯穿"两种合法模板之一
- [ ] 和声有古典色彩点缀（可选但加分）

### 禁止项

- [ ] 使用"红尘 / 江湖 / 苍茫 / 刹那 / 轮回 / 缘分 / 宿命 / 千年 / 流年 / 一梦" 任一 → FAIL
- [ ] 歌词是"我要挥洒人生画卷"这类宏大励志调子 → FAIL
- [ ] 编曲古筝全曲贯穿且同时存在 R&B 鼓点 → FAIL（风格混乱）
- [ ] 歌词用"墨香 / 书香"这种惯用搭配作为意象主体 → FAIL（懒人化）

## 手工评估点

- **留白感**：歌词示例句结束后有没有"还有话没说完"的感觉？
- **具体程度**：每句能不能让人想象出一幅画？
- **东方但不刻板**：一个没有中国风审美的人听了，会不会觉得"哇这很中国"而不是"哦又是古风"？

## 备注

这个 test 是整个 Skill **最容易失败**的一个。因为训练数据里有大量"懒人东方化"的流行歌，模型默认就会往那边滑。
连续测试 3 次都通过才算稳定。
