# Test 01 — 基础流程：毕业前夜 + 钢琴开头

## 用途

验证 Skill 在一个完整清晰的需求下能输出合格的 10 段方案。这是最基础的回归测试，每次修改 Skill 后都要过。

## 输入

```
写一首关于毕业前夜的华语流行歌，钢琴开头，情绪从克制到释怀，带一点轻 R&B 律动和东方画面感，副歌要有大记忆点。
```

## 预期行为

Skill 应当：

1. **不问澄清问题**（信息足够完整）
2. **按 10 段模板**依次输出，每段都有实质内容
3. **内部调用顺序**：song_analyzer → 结构 → 和声 → 旋律 → 歌词 → 编曲 → hook → similarity_guard → de-sim
4. **similarity_guard 跑完**，在第 9 段展示检测结果
5. **至少一项 PASS/WARN 标签**（不能全部隐藏）

## 验收清单

### 必须项（任一不通过 → FAIL）

- [ ] 10 段全部出现，标题按顺序编号
- [ ] 副歌 hook 有明确动机描述（音区 + 节奏 + 歌词锚点）
- [ ] 歌词示例句 ≥ 3 句，全部原创
- [ ] 结构里有非整数 / 非 4 倍数段落（证明没有退化成模板）
- [ ] 第 9 段 similarity_guard 显式给出 PASS/WARN/BLOCK 分类标签
- [ ] 第 10 段要么给改写建议，要么明确说"无需改写"
- [ ] 编曲部分指定了具体乐器 + 进出时机，不是"钢琴进弦乐进鼓"的流水账
- [ ] 情绪最大释放点在 C3，不是 C1

### 应当项（满足 ≥ 4/5 即可）

- [ ] 结构里有 Pre-chorus（钢琴抒情通常有）
- [ ] 使用了至少一个借用和弦或色彩和弦
- [ ] 副歌用长音锚定，不是立刻上一堆十六分音符
- [ ] Bridge 的能量比副歌低，有明显反向
- [ ] 歌词里有"物件锚定抽象情绪"的实例

### 禁止项（任一命中 → FAIL）

- [ ] 歌词示例句出现黑名单词（"陪你到世界尽头" / "永远永远" / "我的心碎" 等）
- [ ] 歌词出现"红尘 / 江湖 / 苍茫"任一
- [ ] 编曲出现"钢琴+弦乐+鼓"这种没信息量的描述
- [ ] Hook 被描述为"纯四度上行再下行"
- [ ] 结构严格是 `V(8) C(8) V(8) C(8) Bridge(8) C(8)`

## 手工评估点

- **旋律可哼度**：副歌 hook 读完后，能不能想象出它的节奏型？
- **意象新鲜度**：歌词示例句里有没有至少 1 句是"没见过的组合"？
- **编曲节奏感**：Arrangement 描述能不能让一个制作人直接开 session？
